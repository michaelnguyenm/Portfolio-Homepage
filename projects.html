Fence Puzzle Game Github

Course: Mobile Computing Spring 2017

  • Created an Android game based on a Professor Layton minigame over the course of ten weeks with a team of three.
  • Developed in Java with Android Studio and uses native Android features.
  • Uses JSON to load in various puzzles from assets. Data such as settings and score is kept persistant across sessions through the use of Android's shared preferences.
  • Implemented vibration and sounds as feedback for the user as well as music.
